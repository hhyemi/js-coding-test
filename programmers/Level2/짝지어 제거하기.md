### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(s) {
  for (let i = 0; i < s.length; i++) {
    if (s.indexOf(s[i] + s[i]) != -1) {
      s = s.replace(s[i] + s[i], '');
      i = 0;
    }
  }

  return s.length == 0 ? 1 : 0;
}
```

맨 처음 글자를 가져와서 반복한다음 해당 반복문자가 있는지 체크해주고 있으면 공백으로 `replace`해준다.  
그런 후 길이로 체크한다.

그러나...  
시간초과로 실패한 테스트이다..  
원인은 `i=0` 인거같다. 앞에는 이미 겹치는 부분이 없는데 0으로 바꿔서 그런거같다..

그래서 그 부분을 수정해줬다.

```javascript
function solution(s) {
  for (let i = 0; i < s.length; i++) {
    if (s.indexOf(s[i] + s[i]) != -1) {
      s = s.replace(s[i] + s[i], '');
      i = s.indexOf(s[i] + s[i]) - 1;
    }
  }

  return s.length == 0 ? 1 : 0;
}
```

정확성은 합격했는데 효율성에서 시간초과다.ㅠㅠ  
찾아보니 replace는 전체 문장을 검사하기때문에 효율성에서 걸리는것 같다.

```javascript
function solution(s) {
  const stack = [];

  for (let i = 0; i < s.length; i++) {
    if (stack[stack.length - 1] == s[i]) {
      stack.pop();
    } else {
      stack.push(s[i]);
    }
  }

  return stack.length == 0 ? 1 : 0;
}
```

stack의 성질을 이용해서 해결하였다 속도도 훨씬 빠르다!
