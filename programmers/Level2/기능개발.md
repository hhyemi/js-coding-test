### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(progresses, speeds) {
  let result = [];

  while (progresses.length > 0) {
    progresses = progresses.map((e, i) => e + speeds[i]);
    let count = 0;

    for (let i = 0; i < progresses.length; i++) {
      if (progresses[i] < 100) {
        break;
      }
      progresses.shift();
      speeds.shift();
      i--;
      count++;
    }

    if (count != 0) {
      result.push(count);
    }
  }

  return result;
}
```

map을 이용하여 각 speeds의 값을 더해준다.  
for문을 돌려 100의 값이 있으면 `shift()`해준다. 여기서 배열이 잘렸기 때문에 `i--`를 해줘야한다.  
잘린 경우 count를 올려줘서 for문이 끝난 후 result에 넣어준다.  
배열의 값이 100보다 작은 경우 for문을 벗어난다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/42586/solution_groups?language=javascript

```javascript
function solution(progresses, speeds) {
  let answer = [0];
  let days = progresses.map((progress, index) => Math.ceil((100 - progress) / speeds[index]));
  let maxDay = days[0];

  for (let i = 0, j = 0; i < days.length; i++) {
    if (days[i] <= maxDay) {
      answer[j] += 1;
    } else {
      maxDay = days[i];
      answer[++j] = 1;
    }
  }

  return answer;
}
```

걸리는 시간을 먼저 days로 구해주고 푸는 답안이다..  
미리 시간을 구했기 때문에 while문과 map()이 필요없었다. (시간절약)
