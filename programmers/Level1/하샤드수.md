### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(x) {
  return x % [...(x + '')].reduce((acc, curr) => acc + parseInt(curr), 0)
    ? false
    : true;
}
```

reduce를 이용하여 합을 구해준후 x로 나눈값에 따라 return 해줬다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12947/solution_groups?language=javascript

```javascript
function solution(x) {
  let num = x;
  let sum = 0;
  do {
    sum += x % 10;
    x = Math.floor(x / 10);
  } while (x > 0);

  return !(num % sum);
}
```

속도측면으로 보면 숫자로 푸는게 빠르다.
