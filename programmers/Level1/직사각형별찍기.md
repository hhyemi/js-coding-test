### :heavy_check_mark: 내가 푼 답안

```javascript
process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
  const n = data.split(' ');
  const a = Number(n[0]),
    b = Number(n[1]);

  let text = '';
  for (let i = 0; i < a; i++) {
    text += '*';
  }

  for (let j = 0; j < b; j++) {
    console.log(text);
  }
});
```

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12969/solution_groups?language=javascript

```javascript
process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
  const n = data.split(' ');
  const a = Number(n[0]),
    b = Number(n[1]);
  const row = '*'.repeat(a);
  for (let i = 0; i < b; i++) {
    console.log(row);
  }
});
```

for문 대신 repeat함수를 이용해서 row를 구성해줬다.
