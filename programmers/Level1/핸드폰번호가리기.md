### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(phone_number) {
  return [...phone_number]
    .map((e, i, arr) => (i < arr.length - 4 ? '*' : e))
    .join('');
}
```

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12948/solution_groups?language=javascript

```javascript
function hide_numbers(s) {
  var result = '*'.repeat(s.length - 4) + s.slice(-4);
  return result;
}
```

repeat을 이용해서 `*` 을만들어주고 slice이용해서 뒤에 4개를 잘라줬다.  
내 코드보다 간단해서 좋다.
