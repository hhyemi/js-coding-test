## 약수의 개수와 덧셈

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(left, right) {
  let answer = 0;

  for (let j = left; j <= right; j++) {
    countBool(j) ? (answer += j) : (answer -= j);
  }
  return answer;
}

function countBool(num) {
  let count = 0;
  for (let i = 1; i <= num; i++) {
    if (num % i == 0) count++;
  }
  return count % 2 == 0 ? true : false;
}
```

countBool : 약수가 짝수개인지 홀수개인지 체크 (짝수면 T 홀수면 F)
함수를 따로이용해서 값을 계산해줬다.
