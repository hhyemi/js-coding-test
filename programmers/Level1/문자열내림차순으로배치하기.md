<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(s) {
  var answer = '';
  let arr = [...s];
  arr.sort(function (a, b) {
    if (a.charCodeAt(0) > b.charCodeAt(0)) {
      return -1;
    } else {
      return 1;
    }
  });

  return arr.join('');
}
```

배열로 바꿔서 아스키코드를 비교해서 정렬을 해주었다.

```javascript https://programmers.co.kr/learn/courses/30/lessons/12917/solution_groups?language=javascript
function solution(s) {
  return s
    .split('')
    .sort((a, b) => (a < b ? 1 : -1))
    .join('');
}
```

split을 이용해서 배열로 만들어준 후 삼항연산자..
아스키코드로 변환안해도 문자열비교는 .. 아스키코드대로 되는거였다.. 몰랐다 ㅠㅠ
