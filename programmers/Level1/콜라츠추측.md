### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(num) {
  var answer = 0;

  while (num > 1) {
    if (answer == 500) {
      return -1;
    }
    if (num % 2 == 0) {
      num = num / 2;
    } else {
      num = num * 3 + 1;
    }
    answer++;
  }

  return answer;
}
```

while문을 이용한 답안이다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12943/solution_groups?language=javascript

```javascript
function collatz(num, count = 0) {
  return num == 1
    ? count >= 500
      ? -1
      : count
    : collatz(num % 2 == 0 ? num / 2 : num * 3 + 1, ++count);
}
```

재귀함수를 이용한 답안이다. 속도는 단순while 일때가 빠르다.
