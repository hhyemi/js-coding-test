### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(a, b) {
  let day = 0;
  let dayOn = ["THU", "FRI", "SAT", "SUN", "MON", "TUE", "WED"];

  for (let i = a - 1; i > 0; i--) {
    if (i == 1 || i == 3 || i == 5 || i == 7 || i == 8 || i == 10 || i == 12) {
      day += 31;
    } else if (i == 2) {
      day += 29;
    } else {
      day += 30;
    }
  }

  return dayOn[(day + b) % 7];
}
```

1월1일이 금요일이고 요일은 7개가 있으니까 7로 나눈 나머지로 요일을 결정하였다.  
근데 ` for(let i= a-1 ; i>0 ; i--){` 조건에 a-1을 a로 쓰고 왜 안되나 고생하였다 ㅠㅠ  
5월24일이면 4월까지 더해야되는데 5월까지 더해서 오류...

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12901/solution_groups?language=javascript

```javascript
function solution(a, b) {
  const monthDay = [31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31];
  const weekDay = ["THU", "FRI", "SAT", "SUN", "MON", "TUE", "WED"];

  let days = b;
  for (let i = 0; i < a - 1; i++) days += monthDay[i];

  return weekDay[days % 7];
}
```

요일만 배열에 담을 생각만 했다.. 날짜도 배열에 담으면 훨씬 코드가 깔끔해진다.  
days 에 b값을 먼저 넣는 방법도 참고하면 좋을 것 같다.
