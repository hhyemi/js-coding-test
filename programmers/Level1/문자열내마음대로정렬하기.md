## 문자열 내 마음대로 정렬하기

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(strings, n) {
  strings.sort(function (a, b) {
    if (a[n] > b[n]) return 1;
    if (b[n] > a[n]) return -1;

    if (a > b) return 1;
    if (b > a) return -1;

    return 0;
  });
  return strings;
}
```

sort함수를 이용해서 -1,1,0 에 따라 정렬이 되는 것을 몰라서 답을 참고해서 풀었다.  
sort함수에서 처음 **_if문 2개는 n번째의 문자를 비교_**하는것이고  
나머지 if문 2개는 만약 동일할경우 **_사전적으로 정렬_**하기 위한 조건문이다.  
if문 조건에서 걸리는 것이 없으면 0을 반환한다.

#### sort(a, b)

- 0보다 작은 경우 a를 b보다 낮은 인덱스로 정렬한다.
- 0을 반환하면 a와 b를 변경하지 않는다.
- 0보다 큰 경우, b를 a보다 낮은 인덱스로 정렬한다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12915/solution_groups?language=javascript

```javascript
function solution(strings, n) {
  return strings.sort((s1, s2) =>
    s1[n] === s2[n] ? s1.localeCompare(s2) : s1[n].localeCompare(s2[n])
  );
}
```

localeCompare함수와 sort()함수를 쓴 코드이다.

#### a.localeCompare(b)

- a가 b보다 앞에 있으면 -1, 뒤에 있으면 1, 같으면 0 반환한다.

### :heavy_check_mark: 참고하면 좋은 다른 답안2

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12915/solution_groups?language=javascript

```javascript
function solution(strings, n) {
  return strings.sort((a, b) => {
    const chr1 = a.charAt(n);
    const chr2 = b.charAt(n);

    if (chr1 == chr2) {
      return (a > b) - (a < b);
    } else {
      return (chr1 > chr2) - (chr1 < chr2);
    }
  });
}
```

**_false가 0 true가 1_**을 이용한 코드이다.  
만약 `a >b` 가 1 이고 `a < b`가 0 이면 `return`은 1이되어서 b를 낮은인덱스로 옮긴다.  
if문 4개를 if-else 1개로 줄인것이 마음에든다.
