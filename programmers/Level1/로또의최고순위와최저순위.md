### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(lottos, win_nums) {
  let answerCnt = 0;
  let zeroCnt = 0;

  lottos.sort((a, b) => a - b);
  win_nums.sort((a, b) => a - b);

  if (lottos.join('') == win_nums.join('')) return [1, 1];

  answerCnt = lottos.filter((e) => win_nums.indexOf(e) !== -1).length;
  zeroCnt = lottos.filter((e) => e == 0).length;

  const answer = answerCnt + zeroCnt > 6 ? 6 : answerCnt + zeroCnt;

  return [6 - answer + 1 == 7 ? 6 : 6 - answer + 1, 6 - answerCnt + 1 == 7 ? 6 : 6 - answerCnt + 1];
}
```

순위를 어떻게할지몰라서 고민을 했다.  
지저분하게 삼항연사자를 사용했다..

```javascript
function solution(lottos, win_nums) {
  const rank = [6, 6, 5, 4, 3, 2, 1];

  let minCount = lottos.filter((v) => win_nums.includes(v)).length;
  let zeroCount = lottos.filter((v) => !v).length;

  const maxCount = minCount + zeroCount;

  return [rank[maxCount], rank[minCount]];
}
```

순위를 배열을 이용해서 해줬다.
