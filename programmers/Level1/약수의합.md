## 약수의 합

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  let answer = 0;

  for (let i = 1; i <= n; i++) {
    n % i == 0 ? (answer += i) : answer;
  }

  return answer;
}
```

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12928/solution_groups?language=javascript&type=all

```javascript
function solution(n) {
  var answer = 0;
  let i;
  for (i = 1; i <= Math.sqrt(n); i++) {
    if (!(n % i)) {
      answer += i + n / i;
    }
  }
  i--;
  return i === n / i ? answer - i : answer;
}
```

계산을 줄이기위해 루트를 이용해서 반만 for문을 돌려준후 값에 2개의 값을 더해줬다.
