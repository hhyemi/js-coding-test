### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  var answer = 0;

  for (let i = 2; i <= n; i++) {
    let j = 2;
    let check = false;

    for (let j = 2; j <= i; j++) {
      if (i % j == 0 && j != i) {
        check = true;
        break;
      }
    }

    check ? answer++ : answer;
  }

  return n - 1 - answer;
}
```

시간초과에 떨어졌다..

```javascript
function solution(n) {
  var answer = 0;

  for (let i = 2; i <= n; i++) {
    if (i % 2 != 0) {
      let j = 2;
      let check = false;

      for (let j = 3; j <= Math.sqrt(i); j += 2) {
        if (i % j == 0 && j != i) {
          check = true;
          break;
        }
      }
      check ? answer++ : answer;
    } else {
      answer++;
    }
  }

  return n - answer;
}
```

해결1. 짝수는 무조건 소수가 아니기때문에 `if(i%2 != 0){`문을 통해 처리를 해줬다.  
해결2. 홀수인 경우 sqrt(n)까지 나눠서 나눠떨어지는지 여부 체크하였다.=> 8*2 를 체크하는 것과 2*8을 체크하는 것은 같기때문에 두 수 중 작은 수만 체크하면 되고, 이는 곧 제곱근 (나눌 수 있는 최소 사이즈의 수) 까지만 체크하면 된다는 뜻이다.  
2로 나눠지는것은 체크했기 때문에 홀수만 보기위해서 `j +=2`를 해준다.

```javascript
function solution(n) {
  if (n == 2) return 1;
  let i, j;
  let primes = [0, 0];
  for (i = 2; i <= n; i++) primes[i] = 1;
  for (i = 2; i <= n; i++) {
    if (!primes[i]) continue;
    for (j = i * 2; j <= n; j += i) primes[j] = 0;
  }
  return primes.reduce((acc, cur) => acc + cur);
}
```
