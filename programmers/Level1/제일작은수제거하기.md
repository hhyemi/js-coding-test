## 제일 작은 수 제거하기

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(arr) {
  if (arr.length == 1) return [-1];

  let small = arr[0];

  for (let item of arr) {
    if (small > item) small = item;
  }

  arr.splice(arr.indexOf(small), 1);

  return arr;
}
```

가장작은 수를 for문으로 찾은다음 해당인덱스값을 splice해줬다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12935/solution_groups?language=javascript

```javascript
function solution(arr) {
  arr.splice(arr.indexOf(Math.min(...arr)), 1);
  if (arr.length < 1) return [-1];
  return arr;
}
```

`arr.indexOf(Math.min(...arr))`을하면 알아서 작은값을 찾아내는 것을 몰랐다.. for문을 돌릴필요가없었다.
